# Продолжаем работу над виджетом банковских операций клиента. Добавил 2 функции.

## Функции:
1. Функция
filter_by_state
, которая принимает список словарей и опционально значение для ключа 
state
 (по умолчанию 
'EXECUTED'
). Функция возвращает новый список словарей, содержащий только те словари, у которых ключ 
state
 соответствует указанному значению.
2. Функция 
sort_by_date
, которая принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание). Функция возвращает новый список, отсортированный по дате (
date
).
